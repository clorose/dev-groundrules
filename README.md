# dev-groundrules

- Git alias / config 스니펫

## Git Commit Rules

커밋 메시지 관련 규칙들을 모아둔 섹션입니다.

- 어떤 타입을 쓸지 (Conventional Commit 확장)
- 언제 어떤 이모지/무드 태그를 쓸지
- 커밋 메시지 예시/패턴

모든 커밋 관련 규칙은 아래 문서 하나에 정리합니다:

- [`git/git.md`](git/git.md)  
  → Conventional Commit 타입 정의, Gitmoji, Commit Mood(🥱 `zzz`, 🎮 `fun`, 🛏 `sleep`, 😡 `altf4` 등) 통합 정리
